IMG Crossmedia Studio

About

Website for IMG Crossmedia Studio. I consider this one my most complete front-end develop work. I have coded all the plugins using jquery so they would appear and work like the designers asked.

The designers also didn't have predicted how the responsivity would work, but they asked me to implement it, so rather than using media querries based on commom devices sizes, I tested how the content was resizing to define the best breaking points.
