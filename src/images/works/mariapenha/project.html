Maria da Penha

About

Front end for the first edition of the Maria da Penha's Short Documentary Films Festival. I was really short on time on this one, so I did just the basics. No responsive layout, a third party plugins. They still use my code on the second edition, but they changed the background, that was the thing I liked most in my version. I sliced in 3 different divs so it would fill all the page width in most cases, without resizing the image.

The images bellow show only the background with this effect
